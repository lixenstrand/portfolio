Latest Project

Inquiry tool for steel

This tool is designed to streamline the process of filtering suppliers and contacts for steel inquiries, making it quick and easy to find the right resources. This tool is uniquely developed to support multi-language capabilities, ensuring users can efficiently draft an inquiry email about various steel grades with just the click of a button.

The tool accommodates both manual entry on the website and an integrated function that allows for automatic inquiry generation directly from an Excel calculation tool. This innovative feature ensures a seamless transition from calculation to communication, significantly enhancing the efficiency of the procurement process.

The core components of this tool include its intuitive user interface, multi-language support, and the seamless integration with Excel, all aimed at simplifying the complexities involved in sourcing steel.

  • Python |
  • SQL |
  • HTML |

Offer and order calculator

This project is a meticulously crafted Excel template designed to calculate steel costs, shipping expenses, and more.

Leveraging various APIs, it integrates real-time freight rates into the calculation, ensuring accurate and up-to-date pricing. This innovative tool seamlessly generates comprehensive offers directly within the Fortnox business system, based on the data input into the Excel template.

The integration facilitates a smooth transition from initial cost estimation to the creation of detailed offers, streamlining the entire procurement process. By automating these calculations, the calculator not only saves time but also enhances accuracy and efficiency in offer generation and order processing.

Technologies used include:

  • Python |
  • SQL |
  • VBA |
  • Excel

Internal company website

The project centers around the development of an internal website for a company, designed as a tailor-made, all-in-one system to meet the unique business needs. At its core, the website features a basic CRM (Customer Relationship Management) system that allows for efficient management of customer interactions and data.

Additionally, it incorporates certificate management, facilitating the tracking and organization of necessary compliance documents. A key component of the platform is the transporter registry, which, along with logistics planning capabilities, streamlines the process of managing shipments and logistics operations.

The ultimate goal was to create a customized, all-encompassing system that supports the diverse needs of the company, from customer management to logistics planning, ensuring a more cohesive and efficient workflow.

  • Python |
  • SQL |
