Förfrågningsverktyg för stål

Detta flerspråkiga verktyg förenklar processen för prisförfrågningar till leverantörer, vilket gör att användare snabbt kan få tillgång till personligt färdigställda förfrågningar och enkelt utarbeta förfrågningar via både webbplatsen och Excel (uträkningsverktyg) med ett enda klick.

Verktyget söker igenom historiska data samt tidigare manuellt inmatad information för att ge en omfattande översikt över stålprodukter, inklusive detaljer om dimensioner, kvantiteter och leverantörer.

teknologier som används:

  • Javascript |
  • Python |
  • SQL |
  • HTML |
  • CSS

Offert och order kalkylator

Detta projekt beräknar stål- och fraktkostnader automatiskt genom att utnyttja API:er för realtidsfraktkostnader.

Intern webbapp för Nordic metal trade

Detta projekt är en intern webbapp som har ett anpassat CRM för effektiv hantering av kunddata, certifikatspårning och ett transportregister för logistikoperationer.
